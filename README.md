# HW2's file Servlet Based Student Personal Cabinet

Please create Backend Application using Servlet and JSP technology which will consist of Registration, Authentication and Personal Cabinet page by integration with PostgreSQL using JDBC driver and Connection Pool

1. Registration/Authentication

1.1. User can register with it's own email and password

1.2. After registration user should be moved to Authentication page to go through authentication process with his email and password

2. Personal Cabinet

2.1. User should be able to add his personal information to cabinet. Please create form with input fields for personal information provisioning (such as firstname, lastname, age, city, country, gender)

2.2. User should be able to see list of courses which he could be enrolled to (list of the courses should be stored in database, fetched from database and rendered in page)

2.3. User should have button near by each course to be enrolled to course (This information should be reflected in databases)

Note: All your solutions SHOULD be pushed to remote Git repository 

Total possible points for this assignment is 10.

Assessment criteria is the following:

1.1. Usage of the technologies and techniques covered in Lecture materials and explained by Instructor - 60%

1.2. Clean Code - 20% (project structure, code structure, patterns usage, logging and etc)

1.3. Storage of password in encrypted way - 20%
